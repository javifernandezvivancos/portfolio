Project 4:
2D games coded on JAVA
Here you can find 2 games coded on Java with the purpouse of using all the functionalities that JAVA offers: object oriented programming, interfaces, abstract classes, constructors, visibility of every class, exceptions... To execute them just compile the projects and load the configuration.txt
file. All the feautures of the projects are well documented (in Spanish) in the index.html file of the doc folder.
